The Rules Analyzer tool graphically shows and explains why user requests were allowed or denied. Log files are created by the Application Control agent on the client machine and can then be uploaded and viewed in the Rules Analyzer. This allows system administrators to fine tune their configurations and make sure they function as planned. If there are any discrepancies they can immediately see which rule reacted unexpectedly and what needs to be changed to make the Application Control agent behave as expected.
